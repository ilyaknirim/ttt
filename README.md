# Крестики-Нолики в Telegram Mini App

Простое приложение для игры в крестики-нолики, которое можно запустить как Telegram Mini App. Игру можно поделиться ссылкой в чате, чтобы сыграть с друзьями.

## Особенности

- Игра на двоих: крестики (X) и нолики (O)
- Состояние игры сохраняется в URL для обмена
- Поддержка Telegram Web App API
- Простой интерфейс, адаптированный для мобильных устройств

## Развертывание на GitHub Pages

1. Создайте новый репозиторий на GitHub
2. Загрузите файлы проекта: `index.html`, `styles.css`, `app.js`, `README.md`
3. В настройках репозитория (Settings > Pages) выберите источник "Deploy from a branch"
4. Выберите ветку `main` и папку `/ (root)`
5. Сохраните настройки. GitHub Pages сгенерирует URL вашего приложения (например, `https://username.github.io/repository-name`)

## Настройка Telegram Mini App

1. Создайте бота через @BotFather в Telegram
2. Используйте команду `/newbot` для создания нового бота
3. Получите токен бота
4. В настройках бота установите Web App URL на ваш GitHub Pages URL
5. Добавьте кнопку в меню бота или используйте inline режим

## Как играть

1. Откройте приложение в Telegram
2. Нажмите "Новая игра" для начала
3. Делайте ходы, кликая на клетки
4. Нажмите "Поделиться игрой" для копирования ссылки
5. Отправьте ссылку другу, чтобы он присоединился к игре
6. Играйте по очереди, обновляя страницу после каждого хода
